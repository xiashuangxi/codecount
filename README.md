# codecount
显示`Ada`源文件的代码行数。

# 编译
`gprbuild gant/CodeCount.GPR`

# 运行
```
> cc.exe cc.adb  code_count11.adb

File Name: cc.adb
Line Count: 28
Code Count: 22
Comment Count: 1
Null Count: 5
File Name: code_count11.adb
Line Count: 91
Code Count: 70
Comment Count: 9
Null Count: 12

Total Line Count: 119
Total Code Count: 92
Total Comment Count: 10
Total Null Count: 17
```
